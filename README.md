# Spring Cloud, Eureka를 이용한 간단한 MSA 실습

## 웹 쇼핑몰 사이트
> MSA의 동작위주의 실습  
> MSA의 동작 학습/응용이 목적이기 때문에 상세한 기능 구현없이 ResponseEntity로 데이터만 출력  
> 사용한 기술 : 
>* Spring Boot  
>* Spring Cloud
>* Eureka
>* H2 DB
